
Projeto de Tradutor de Artigos Técnicos utilizando o Azure AI Translator.

A aplicação realiza a tradução automática de textos técnicos entre diferentes idiomas, utilizando recursos de Processamento de Linguagem Natural (NLP) oferecidos pelo Microsoft Azure. O serviço é capaz de detectar automaticamente o idioma de origem e gerar traduções mantendo o contexto e a terminologia técnica.

O projeto foi desenvolvido com integração direta à API REST do Azure AI Translator, aplicando boas práticas de organização de código, segurança no uso de chaves de API e documentação clara para facilitar a manutenção e evolução da solução.

Principais recursos utilizados:
- Azure AI Translator
- Tradução automática de textos técnicos
- Detecção automática de idioma
- API REST
- Linguagem Python
- Versionamento com GitHub

Este projeto demonstra, na prática, o uso de serviços cognitivos do Azure aplicados a um cenário real, sendo voltado para aprendizado e apresentação de habilidades em IA, computação em nuvem e integração de APIs.
