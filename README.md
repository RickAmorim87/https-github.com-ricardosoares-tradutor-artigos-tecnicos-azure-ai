# https-github.com-ricardosoares-tradutor-artigos-tecnicos-azure-ai
Este projeto consiste em um tradutor de artigos técnicos utilizando os serviços de Linguagem de IA do Microsoft Azure.  A aplicação permite traduzir textos técnicos entre diferentes idiomas, preservando o contexto e a terminologia técnica, por meio do serviço Azure AI Translator.
